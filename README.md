# Projeto de Portfólio Pessoal - Maximiliano Shimabucuro

## Introdução

## 1. Visão Geral do Projeto

Este projeto consiste na criação de uma landing page de portfólio pessoal, desenvolvida como atividade prática extra para o curso "Introdução à Programação Front-End e Back-End" do programa SCTEC. A página foi projetada para funcionar como um cartão de visitas digital, apresentando de forma clara e organizada minhas informações, competências e projetos.

O principal objetivo é aplicar os conhecimentos fundamentais de desenvolvimento web adquiridos no curso, criando um produto final que seja funcional, esteticamente agradável e que possa ser expandido conforme avanço na minha carreira em tecnologia. A estrutura da página foi dividida em seções estratégicas para facilitar a navegação: uma introdução pessoal, uma lista de habilidades técnicas e comportamentais, e uma galeria de projetos.

## 2. Tecnologias Utilizadas

O desenvolvimento foi baseado no tripé essencial do front-end, garantindo a conformidade com os requisitos da atividade e as boas práticas de mercado:

*   **HTML5:** A estrutura do site foi construída com HTML5, utilizando tags semânticas (`<header>`, `<main>`, `<section>`, `<footer>`, etc.) para garantir um código limpo, acessível e otimizado para motores de busca (SEO).

*   **CSS3:** A estilização foi feita com CSS3. Foi implementado um design responsivo utilizando Flexbox e Media Queries, que adaptam o layout para diferentes tamanhos de tela (desktop e mobile). Além disso, foram adicionadas transições suaves e uma funcionalidade de **tema claro/escuro (dark mode)** para melhorar a experiência do usuário (UX).

*   **JavaScript (ES6+):** A interatividade da página foi implementada com JavaScript. A principal funcionalidade é a troca de tema, que manipula o DOM adicionando ou removendo uma classe do `<body>` e salva a preferência do usuário no `localStorage` do navegador, fazendo com que a escolha seja mantida entre as visitas. O código foi escrito de forma não obstrusiva, aguardando o carregamento completo do DOM antes de ser executado.

## 3. Estrutura e Seções

A página está dividida nas seguintes seções:

1.  **Cabeçalho (`<header>`):** Contém o nome, o título/cargo e o botão de interação para troca de tema.
2.  **Sobre Mim (`<section id="sobre">`):** Uma breve descrição sobre minha trajetória, paixões e objetivos profissionais.
3.  **Habilidades (`<section id="habilidades">`):** Dividida em "Habilidades Técnicas" e "Habilidades Comportamentais", listando as competências mais relevantes.
4.  **Projetos (`<section id="projetos">`):** Uma galeria com cards para cada projeto, contendo nome, descrição, tecnologias utilizadas e links relevantes (demonstração ou repositório).
5.  **Rodapé (`<footer>`):** Informações de copyright.

## 4. Como Executar

Este é um projeto front-end estático. Para visualizá-lo, não é necessário nenhum ambiente de servidor complexo.

1.  Faça o download ou clone o repositório.
2.  Descompacte a pasta do projeto.
3.  Abra o arquivo `index.html` em qualquer navegador de internet moderno (Google Chrome, Mozilla Firefox, etc.).
